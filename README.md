# gifter

## Build package:
Build package:
```
ant
```
Run test:
```
ant test
```
Run example:
```
ant run
```
Build Jar file:
```
ant dist
```

## Command line example
```
java -classpath /home/jay/ws/gifter/dist/lib/Gifter-20160327.jar com.cooldoger.gifter.Main tst/res/example.txt -nonFamilyGifter
```
